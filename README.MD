# Getting Started

1. **Install Dependencies:**

    Ensure you have [Node.js](https://nodejs.org/) installed. Then run:

    ```bash
    npm install
    ```

2. **Run the Project:**

    Start the development server with:

    ```bash
    npm start
    ```

3. **Build for Production:**

    Compile the project for production with:

    ```bash
    npm run build
    ```
